创建此博客步骤 0x00 新建的博客文件夹 hugo new site /path/to/site

0x01 新建自己博客的md文件 hugo new about.md

0x02 下载博客主题 git clone https://github.com/spf13/hyde.git

0x03 本地启动博客 hugo server –theme=m10c –buildDrafts

0x04 在github上部署博客 注意：现在github创建仓库后再用下面的命令。

hugo –theme=m10c baseUrl=“https://CAmateur.github.io/" –buildDrafts

此命令执行后会自动生成public文件。

0x05 初始化git仓库 git init

0x06 在git中设置远程仓库地址 git remote add origin 仓库地址

0x07 添加到git缓存区 git add -A

0x08 添加到git本地区 git commit -m “备注内容”

0x09 推送到远程仓库地址 git push origin master

0x0A 你的博客已经可以访问啦!

提示: 0x00 将更改后的文件推送到远程仓库后，进入博客没有更新。

这种情况需要进入你的github仓库里面刷新一下

参考网站 hugo中文文档 https://www.gohugo.org/
